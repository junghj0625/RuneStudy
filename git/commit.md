
# What is a Commit?

- 스테이징된 변경 사항을 스냅샷으로 저장하는 작업.
- 커밋 시 **Blob**, **Tree**, **Commit** 오브젝트가 생성됨.
- 커밋이 누적되면 수정된 오브젝트 또한 누적됨.
- GC 과정을 통해 Packfile로 정리됨. (최신 오브젝트 + Delta)

# Blob (Binary Large Object)

- 바이너리 데이터로 압축된 프로젝트 파일. (헤더 + 내용)
- 파일이 수정될 경우 새로 생성되고 저장됨. 해시값으로 식별됨.

# Tree

- 파일 이름, 폴더 구조 등의 메타데이터를 포함.

# Commit

- Tree 해시값, 부모 Tree 해시값, 작성자 등의 커밋 관련 데이터를 포함.
