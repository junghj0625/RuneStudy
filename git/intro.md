
# What is Git?

- 가장 대중적인 버전 관리 시스템.

# Concepts

- Repository : Git으로 트래킹되는 프로젝트 폴더.
- Clone : 원격 리포지토리를 로컬로 복사하는 작업.
- Staging : Git에 변경 사항을 지정하는 작업.
- [commit](commit.md) : 스테이징된 변경 사항을 스냅샷으로 저장하는 작업.
- [branch](branch.md) : 독립적으로 관리되는 워크플로우.

# Appendix

- [checkout](checkout.md) : 특정 커밋 또는 브랜치로 이동하는 작업.
