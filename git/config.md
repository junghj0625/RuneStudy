
# Config

- Git 설정.

# Commands

```Bash

# Configuration Levels
# System (all users): --system
# Global (current user): --global
# Local (current repo): --local
# Delete : --unset

git config --global user.name "Your Name" # 사용자 이름
git config --global user.email "you@example.com" # 사용자 이메일
git config --global core.ignorecase false # 경로명 대소문자 구분

```
