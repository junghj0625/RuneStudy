
# Config

- Git 설정.

# Commands

```Bash

# Configuration Levels
# System (all users): git config --system
# Global (current user): git config --global
# Local (current repo): git config --local

git config --global user.name "Your Name" # 사용자 이름
git config --global user.email "you@example.com" # 사용자 이메일

```
